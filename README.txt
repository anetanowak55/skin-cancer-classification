W tym katalogu znajdują się: 
1) przykładowe dane oraz plik example.ipynb - w celu sprawdzenia, jak model klasyfikuje przykładowe obrazy
2) katalog sample-dataset - zawiera przykładowy zbiór danych
3) katalog scripts - zawiera skrypty użyte do wygenerowania modeli, trenowania oraz pomocnicze
4) plik wideo example.mp4 - przedstawia wykonanie klasyfikacji dla wybranego obrazu

Do uruchomienia plików IPYNB potrzebne są:
- Python (w moim przypadku 3.10.4)
- biblioteki:

pip install tensorflow==1.8
pip install keras
pip install pandas
pip install numpy
pip install matplotlib
pip install scikit-learn

- platforma Jupyter Notebook (pip install notebook)

Następnie wystarczy otworzyć w terminalu katalog w którym znajduje się plik IPYNB oraz wpisać komendę "jupyter notebook".
